# RentCore vs Competitors — Honest Comparison
**Date:** April 13, 2026
**Competitors:** BelongHome.com, Side.com

---

## The Brutal Truth

RentCore looks like a **Tailwind template**. BelongHome and Side look like **brands**.
The difference is NOT code quality — it's design personality, visual storytelling, and emotional impact.

---

## Top 10 Things They Have That We Don't

### 1. REAL PHOTOGRAPHY
- BelongHome: City skyline hero, real video testimonials
- Side: Multi-column parallax photo grid
- **Us: Zero real property/lifestyle photos. No hero image. No visual storytelling.**

### 2. BRAND PERSONALITY
- BelongHome: Playful SVG illustrations (clouds, balloons) — instantly memorable
- Side: Warm mauve/tan palette — feels unique, not template
- **Us: Blue + orange = the most generic SaaS palette possible. Nothing memorable.**

### 3. VISUAL VARIETY BETWEEN SECTIONS
- BelongHome: Each section has different layout, bg treatment, visual style
- Side: Alternating light/dark, parallax layers, stat circles, different column layouts
- **Us: Every section = white bg + orange label + heading + card grid. Repetitive.**

### 4. SCROLL-LINKED ANIMATIONS
- BelongHome: Elements animate differently based on scroll position
- Side: Parallax transforms (images move at different speeds), scale reveals
- **Us: Only fade-up. Every element does the same animation. No parallax.**

### 5. VIDEO CONTENT
- BelongHome: 5 video testimonials from real customers
- Side: Background video in hero
- **Us: Zero video anywhere. No hero video, no testimonials.**

### 6. MEANINGFUL SOCIAL PROOF
- BelongHome: Google review badge, video testimonials with names/faces, triple guarantee
- Side: Partner logos, growth stats, real data
- **Us: Colored circles with initials "DK MJ RS LP" — feels fake. "4.9 from 120+" — no source.**

### 7. HERO VISUAL WEIGHT
- BelongHome: City photo + floating illustrations + large headline
- Side: Full-width cinematic video + dark overlay
- **Us: Dot grid + barely-visible orbs. Hero feels empty despite the fixes.**

### 8. TYPOGRAPHY SOPHISTICATION
- Side: Light body weight (300) creates elegant contrast with bold headings
- BelongHome: Clean hierarchy with generous line-height
- **Us: Everything is regular/medium weight. No typographic elegance.**

### 9. INTERACTIVE ELEMENTS
- BelongHome: Inline rent calculator, rotating value props
- Side: Hover underline animations, accordion with custom styling, animated charts
- **Us: Pricing calculator works but is buried. Hero form is static.**

### 10. EMOTIONAL STORYTELLING
- BelongHome: "Made Loveable" — tells a story of caring
- Side: "#1 Platform for Top Agents" — tells a story of ambition
- **Us: "Stop Managing. Start Earning." — tells nothing about WHO we are**

## What We CAN Fix (Without Full Redesign)

### TIER 1: Biggest Impact, Doable Now

**A. Add a hero background image**
- Real photo of a Phoenix/Arizona property or skyline
- Dark or light overlay so text remains readable
- Kills the "empty template" feel instantly
- Reference: BelongHome's city photo approach

**B. Add visual variety between sections**
- Not every section should be white + card grid
- Alternate: full-width image sections, split layouts, dark sections
- We already have Services (dark) and AreasServed (dark) — need MORE variety
- WhyRentCore could be a split layout (image + text) instead of 3 cards
- HowItWorks could have a vertical timeline instead of 3 columns

**C. Add parallax/scroll-linked animations**
- GSAP ScrollTrigger with scrub (already installed, not using scrub)
- Different elements move at different speeds on scroll
- Hero headline could have subtle parallax offset
- Background orbs could move with scroll
- Cards could stagger-reveal instead of all fading in at once

**D. Add real property photos throughout**
- WhyRentCore: Add a property image alongside the cards
- About: Dale's photo is good — add 2-3 property photos around it
- Between sections: Full-width property photo bands as visual breakers

**E. Improve social proof**
- Replace initial circles with Google Reviews widget or real review text
- Add 2-3 text testimonials with names and roles
- Add a "Google Reviews" badge with actual rating
- Add "Licensed Arizona Broker #XXXXX" prominently

### TIER 2: Medium Impact

**F. Typography refinement**
- Body text: Try weight 300 (light) — creates elegance like Side.com
- Increase heading sizes slightly
- Add letter-spacing variation between sections

**G. Color refinement**
- Add a warm accent (like Side's mauve or a warm sage) to break blue/orange monotony
- Use color shifts between sections (not just white everywhere)

**H. Micro-interactions**
- Nav links: animated underline on hover (Side.com pattern)
- Cards: tilt effect on hover (not just translateY)
- Buttons: icon shift on hover (already doing this with ArrowRight)

### TIER 3: Nice to Have

**I. Hero value-prop rotation**
- Like BelongHome's rotating text: "With guaranteed rent. With vetted tenants..."
- Could rotate through Dale's key messaging points

**J. Section layout variety**
- WhyRentCore: image-left, text-right split instead of 3 cards
- HowItWorks: vertical timeline with alternating left/right
- Add a full-width testimonial quote section between existing sections

---

## Recommended Priority Order

1. **Hero background image** — #1 impact, kills empty vibe
2. **Real property photos** — throughout site, kills template feel
3. **Parallax animations** — scroll-linked GSAP with scrub
4. **Section layout variety** — break the card-grid-on-white pattern
5. **Social proof upgrade** — real testimonials, Google badge
6. **Typography light weight** — body text 300
7. **Micro-interactions** — nav underlines, card tilts
8. **Color accent** — warm third color

---

## What This Is NOT

This is NOT a full redesign. This is a **visual upgrade** using the same components and content. The structure and copy are fine — the visual execution needs to match the level of BelongHome and Side.
